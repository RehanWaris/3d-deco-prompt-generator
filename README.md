# 3D Deco Installation Prompt Generator

A high-fidelity prompt generator built with React and Tailwind CSS, designed to help you create stunning 3D renditions of decorative installations using AI models like Midjourney, DALL-E 3, and Stable Diffusion.

## Features
- **Artistic Style Selection**: Choose from various styles like Art Deco, Bio-Futurism, and more.
- **Material & Lighting**: Fine-tune the physical properties and atmosphere of the installation.
- **Hardware Intelligence**: Recommends the best AI model and GPU based on the complexity of your design.
- **One-Click Copy**: Easily copy the generated prompt for use in your favorite AI generator.

## Tech Stack
- React
- Vite
- Tailwind CSS (via CDN)
- Lucide React (Icons)

## Getting Started
To run this project locally:
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
